# Brand Guide — Senior Project Tracker

## 1. ระบบนี้คืออะไร (1 บรรทัด)
ระบบติดตามงาน Senior Project รายบุคคล ที่เตือนนักศึกษาให้ทำงานตรงตามกำหนดการในวิชา ไม่ใช่ระบบจับผิดหรือลงโทษ

## 2. บุคลิกแบรนด์ (Brand Personality)
ให้นึกถึง **"เพื่อนที่คอยเตือนด้วยความหวังดี"** ไม่ใช่ "อาจารย์ที่จับผิด" หรือ "ระบบราชการที่เคร่งเครียด"

| ลักษณะ | คำอธิบาย |
|---|---|
| ทันสมัย (Modern) | ดีไซน์เรียบ สะอาด ไม่ใช้ลายเส้น/ไอคอนแบบเก่า |
| อบอุ่น (Warm) | สีพาสเทล โทนนุ่มนวล ไม่ใช้สีแดง/ดำเข้มแบบ "เตือนภัย" |
| สบายตา ไม่เครียด (Calm) | ลด pressure ทางสายตา แม้เนื้อหาจะเป็นเรื่อง deadline |
| ตรงไปตรงมา (Clear) | บอกสถานะงานชัดเจน ไม่ใช้ศัพท์ยาก ไม่ใช้คำสั่งห้วนๆ |

## 3. น้ำเสียงการสื่อสาร (Voice & Tone)

### โทนที่ใช้
- พูดแบบให้กำลังใจ ชวนลงมือทำ ไม่ใช่สั่งหรือตำหนิ
- ใช้ "คุณ" หรือชื่อนักศึกษา ไม่ใช่ "นักศึกษาคนนี้" หรือคำสรรพนามที่ดูเป็นทางการเกินไป
- ข้อความแจ้งเตือนเน้น "เหลือเวลา / ควรทำอะไรต่อ" มากกว่า "คุณยังไม่ทำ"

### ตัวอย่างข้อความ

| สถานการณ์ | ❌ ไม่ควรใช้ | ✅ ควรใช้ |
|---|---|---|
| เตือนก่อน deadline | "คุณยังไม่ส่งงาน ห้ามลืม!" | "อีก 2 วันถึงกำหนดส่งงาน 'บทที่ 3' แล้วนะ 😊 พร้อมหรือยัง?" |
| งานเลยกำหนด | "คุณส่งงานล่าช้า ผิดกฎ" | "งาน 'บทที่ 3' เลยกำหนดมาแล้ว ถ้ามีปัญหาอะไร ทักอาจารย์ที่ปรึกษาได้เลยนะ" |
| ส่งงานสำเร็จ | (ไม่มีข้อความ) | "เยี่ยมมาก! ส่งงาน 'บทที่ 3' แล้ว 🎉 งานต่อไปคือ..." |

## 4. กลุ่มผู้ใช้และสิ่งที่แต่ละกลุ่มอยากรู้สึก

- **นักศึกษา**: อยากรู้สึกว่า "มีคนช่วยเตือน ไม่ใช่มีคนจับตาดู"
- **อาจารย์ที่ปรึกษา**: อยากรู้สึกว่า "เห็นภาพรวมลูกศิษย์ได้ไว ไม่ต้องไล่ทวงเอง"
- **อาจารย์ผู้สอน (Admin)**: อยากรู้สึกว่า "คุมภาพรวมทั้งวิชาได้ มีข้อมูลตัดสินใจ"

## 5. ภาษา
ใช้ภาษาไทยและอังกฤษแบบสลับกันได้เต็มรูปแบบ (bilingual switchable) — UI label และข้อความระบบควรมีคำแปลทั้งสองภาษาเตรียมพร้อมไว้ ไม่ตายตัวภาษาเดียว
